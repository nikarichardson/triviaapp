An app that displays questions: both all questions abd by categories. Questions show the question, category, and difficulty rating by default and allow for display of the answer upon click. 

Questions can be deleted and added. 

All questions can be searched based on a text query string. 

A quiz game randomizes all questions or within a specific category and reports the visitor's score of correct answers. 

Tests for the database are included in the backend `test_flaskr.py` file.
